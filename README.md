# porters-solver
A tool to optimize business decisions for profit using constraints.

# Requirements

- scikit-learn
- python3.x
- scipy
- pandas
- numpy

# Quickstart

For normal-sized datasets (n>10000), run:

```python
cd porters-solver
python porters-analysis.py
```

For tiny datasets (n<200), run:

```python
cd porters-solver
python porters-analysis-light.py
```
